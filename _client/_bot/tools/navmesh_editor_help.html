Navmesh Editor

Интерактивный редактор навигационного меша поверх радара CS2

# основная команда python tools/navmesh_editor.py --map de_poseidon # с параметрами python tools/navmesh_editor.py --map de_nuke --scale 3.0 --mode competitive
🔀 Режимы
M
MOVE — перемещение (по умолчанию) Выбирать, перетаскивать и удалять существующие узлы.
C
CREATE — создание узла ЛКМ на пустом месте — новый узел. ЛКМ на узле — выбрать его.
G
EDGE — добавить ребро Клик на узел A, затем клик на узел B → двустороннее ребро. Вес считается автоматически.
L
LOOK — направление взгляда для A->B Выбери узел A, затем B. ЛКМ в пустое место задаёт look_point, клавиша A задаёт fixed_yaw.
ESC
Вернуться в MOVE В режимах EDGE/LOOK: первый ESC отменяет текущий выбор, следующий — переключает в MOVE.
🖱️ Мышь
ЛКМ на узлеВыбрать узел (MOVE / CREATE)
ЛКМ + тащитьПереместить узел (MOVE). Веса рёбер пересчитаются.
ЛКМ на пустоСоздать новый узел (CREATE)
ЛКМ → ЛКМСоединить два узла ребром (EDGE) или выбрать A->B (LOOK)
ЛКМ в пустоНазначить look_point для выбранного A->B (LOOK)
ПКМ на узлеУдалить узел и все его рёбра — в любом режиме

⌨️ Клавиши
M / C / G / L Переключить режим
ESC Вернуться в MOVE
D Удалить выбранный (или под курсором) узел — любой режим
A Назначить/сменить action; в LOOK — fixed_yaw для выбранного A->B
K Удалить look-настройку для выбранного A->B (LOOK)
X Удалить ребро между выбранным узлом и узлом под курсором
U Undo — отменить последнее действие (до 100 шагов)
S Сохранить navmesh/map.json
N Показать / скрыть номера узлов
E Показать / скрыть рёбра
Q Выход (при несохранённых — нажать Q ещё раз или S)
⚙️ Значения Action (назначается клавишей A)
(пусто) Обычное движение без дополнительных действий
jump Одиночный прыжок при достижении узла
crouch Приседать (Ctrl зажат) до следующего узла
shift Идти тихо (Shift зажат) до следующего узла
look_north Повернуть взгляд на север
look_south Повернуть взгляд на юг
look_east Повернуть взгляд на восток
look_west Повернуть взгляд на запад
💻 Команды запуска
🗺️ Редактор навмеша
Базовый запуск python tools/navmesh_editor.py --map de_poseidon
Крупнее / мельче python tools/navmesh_editor.py --map de_poseidon --scale 5.0
Режим игры python tools/navmesh_editor.py --map de_nuke --mode competitive
Все параметры python tools/navmesh_editor.py --map de_poseidon --scale 4.0 --mode wingman
--mapназвание карты (de_poseidon, de_nuke, …)
--scaleмасштаб окна, по умолчанию 4.0
--modewingman (по умолч.) или competitive
🤖 Запуск бота (nav_demo.py)
Просто бежать python nav_demo.py --map de_poseidon
Движение + взгляд (edge look) python nav_demo.py --map de_poseidon --control-mode move+look
Только модуль мыши python nav_demo.py --map de_poseidon --control-mode look_only
К конкретному узлу python nav_demo.py --map de_poseidon --target 42
Чтение из памяти python nav_demo.py --map de_poseidon --reader memory
Запись маршрута python nav_demo.py --map de_poseidon --record
Запись в файл python nav_demo.py --map de_poseidon --record --record-out my_path.json
--mapназвание карты
--readerradar (по умолч.) / memory
--targetID узла назначения (смотреть в редакторе, N)
--recordвключить запись waypoints
--record-outпуть для сохранения waypoints (JSON)
--record-dedupмин. расстояние между точками (по умолч. 32)
--control-modemove+look (по умолч.) или look_only
