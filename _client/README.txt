Содержимое этой папки — всё, что нужно на VM для vm_agent.

Скопируйте файлы на VM (например в C:\CS2_FARM), затем:
  npm install
  python vm_agent.py
или run_agent.bat

Папку FSM_PANEL для работы агента не нужна: AutoIt DLL и cs2_machine_convars.vcfg уже здесь; fsm.cfg для CS2 лежит в settings (опционально, exec вручную при необходимости).
